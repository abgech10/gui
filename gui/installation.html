1.3.4 Installation

Installing PySide6

  1. Open Command Prompt (CMD)
  2. Run the following command to install PySide6: 
    pip install PySide6

    Wait for the installation to complete.

  3. Verify the installation:

    Run the following command to check if PySide6 is installed:

    pip show PySide6

    If installed, you will see details like version and location.

    If not, ensure your Python and pip are correctly installed and try again.

Troubleshooting

  • If pip is not recognized, ensure Python and pip are added to your system's PATH.
  • Try using python -m pip install PySide6 if pip alone doesn't work.
