1.3 GUI DEVELOPMENT

1.3.1 Some GUI in Python:

  • PyQt - Powerful, feature-rich, and best for professional applications. Uses Qt framework.
  • Tkinter - Comes built-in with Python, easy to use, but limited in design.
  • Kivy - Best for touch-screen and mobile applications.
  • PySide - Similar to PyQt but with a more flexible licensing model.
  • PyGTK - Good for Linux and GNOME applications.
  • Pywebview - If you want to build a desktop app using web technologies.
  • DearPyGui - Fast and modern for game-like UIs with GPU acceleration.

We choose PySide6: which is a better version of PySide.
